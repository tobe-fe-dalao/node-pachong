# node-pachong
一个低门槛，可快速上手的Node爬虫脚手架，对JavaScript下的前端友好

[使用文档介绍（1-10弹）](https://juejin.cn/post/7134635146350690318)  |  [博文介绍]()  
 

# 特性

- ✅简洁，易用，快速启动
- ✅对前端友好，小白可快速上手
- ✅封装常见工具链，more is less
- ✅Axios/Puppeteer/Mysql/assert/Fastool
- ✅更多技术封装等你挖掘~
